# ALX Listing App

## Project Description
The ALX Listing App is a modern Airbnb clone built as part of the ALX Pro Dev curriculum. The project focuses on building a high-performance, responsive property listing page using Next.js and Tailwind CSS.

## Project Goals
- Scaffold a scalable Next.js application with TypeScript and Tailwind CSS.
- Establish a clean, modular folder structure.
- Create reusable UI components for property listings.
- Implement responsive design principles.

## Folder Structure
- `components/common/`: Reusable UI components like Button and Card.
- `interfaces/`: TypeScript interfaces for type safety.
- `constants/`: Global constants and configuration.
- `public/assets/`: Static assets including images and SVGs.
- `pages/`: Application pages using the Next.js Pages Router.
- `styles/`: Global CSS and Tailwind configurations.

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd alx-listing-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
